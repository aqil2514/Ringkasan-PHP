1.1 Dasar-dasar PHP (Sintaks)

Sama dengan HTML, PHP memilikis sintaks dasar untuk membuka dan menutup kode, yaitu dengan "<?php" untuk membuka dan "?>" untuk menutup, contoh:


<?php echo "ini adalah kode di dalam PHP" ?>



Code tersebut masih bisa disederhanakan menjadi berikut


<?= echo "ini adalah kode di dalam PHP" ?>


1.2 Dasar-dasar PHP (Variabel)

Untuk menambahkan suatu data dalam variabel, dapat dibuat dengan tanda dolar, contoh.

$nama = "Muhamad Aqil Maulana";

Variabel tidak boleh diawali dengan angka. Namun, boleh mengandung unsur angka.

Cara memanggilnya, cukup buka tag PHP dan langsung tulis nama variabelnya sesuai dengan yang ditentukan.

<?php echo $nama?>

1.3 Dasar-dasar PHP (Penggabungan dua string)

Penggabungan dua string dalam PHP menggunakan tanda titik. Contoh, saya memliki tiga variabel.

$nama_depan= "Muhamad";

$nama_tengah= "Aqil";

$nama_belakang= "Maulana";


Di bawah ini adalah contoh pemanggilannya.

$nama_depan= "Muhamad" . " " $nama_tengah= "Aqil" . " ". $nama_belakang= "Maulana"

1.5 Dasar-dasar PHP (Komentar)

Ada dua cara memberika komentar dalam PHP, yaitu dengan tanda kurung dua kali "//" dan tanda bintang ditambah tanda kurung "/*...*/", contoh:

// Ini adalah komentar dalam PHP dengan menggunakan dua kali tanda kurung.


/* Ini adalah komentar dalam PHP dengan menggunakan tanda bintang + tanda kurung. Mudah kan?*/

1.6 Dasar-dasar PHP (echo, print, print_r, var_dump)

Ada 4 cara untuk menampilkan suatu tulisan ke dalam web, yaitu dengan menggunakan echo, print, print_r, dan var_dump

echo "Muhamad Aqil Maulana"


Nama Fungsi Contoh

2.1. Control Flow


Dalam konteks ini, tidak beda jauh antara Javascript dengan PHP. PHP memiliki 2 jenis Control Flow, yaitu pengulangan dan pengkondisian. Berikut adalah Keywordnya:

  1. for
  2. while
  3. do...while
  4. if...else

2.2. Control Flow (for)


For digunakan untuk menentukan berapa kali suatu kode itu dijalankan. Keyword ini seringkali digunakan untuk membuat kode yang berulang-ulang. format:

for ($nilai_awal; $nilai_awal </> nilai akhir; increment / decrement )


Contoh:

<?php

for ($x = 0; $x < 10; $x++ ){

echo "The number is: $x <br>";

}

?>


Penjelasan:

$x = 0; // Variabel awal adalah 0

$x < 10; // Variabel akhir adalah 10

$x++ // Increment

2.3. Control Flow (While)


While adalah sebuah jenis control flow yang akan menjalankan serangkaian kode apabila kondisi di dalam tanda kurung itu bernilai true. Format

while (kondisi)


Contoh :

while ($i < 5){

echo "Hello World <br>";

$i++ // Increment

}

2.4. Control Flow (Do...While)

Do...While adalah salah satu control flow PHP yang berfungsi untuk melakukan looping juga. Bedanya dengan While, Do While akan melakukan minimal satu kali pengerjaan sedangkan while tidak.

Sintaks:

Do{

Aksi

} while(kondisi)


Contoh:

$i = 0;

do{

echo "Hello World <br>";

$i++

} while($i < 5 )

2.5. Control Flow (If...Else)

Sama seperti yang lainnya, If...Else adalah looping yang ada pada PHP. Tidak hanya If...Else, PHP juga mengenal istilah elseif. Penulisannya bisa disambung (elseif) dan bisa dipisah (else if).

Sintaks:

if(Kondisi) {

Aksi

} elseif(Kondisi) { // Juga bisa ditulis else if

Aksi

} else{Aksi}

3.1. Function

Sama seperti Javascript, PHP juga mengenal function. Dalam PHP, ada dua jenis Function, yaitu:

  1. Built In Function : Function yang telah tersedia oleh PHP secara default dapat digunakan secara bebas.
  2. User Defined Function : Function yang dibuat sendiri.

Di bawah ini adalah beberapa Built In Function:

  • String

    1. .strlen() = Menghitung panjang string
    2. .strcmp() = Menggabungkan dua buah string
    3. .explode() = Menggabungkan kedua string menjadi array
    4. .htmlspecialchars() = Memberikan keamanan pada website

    Utility

    1. .isset() = Apakah variabel telah dibuat? (Hasil boolean)
    2. .empty() = Variabel apa yang kosong?
    3. .die() = Memberhentikan program
    4. .sleep() = Memberhentikan sementara program

    Date / Tiime

    1. .time()
    2. .date()
    3. .mktime()
    4. .strtotime()
    5. Lain-lain

3.2. User Defined Function

Dalam PHP, programmer bisa membuat functionnya tersendiri. Programmer bisa menggunakan format seperti di bawah ini.

function salam($nama = "Admin", $waktu = "Datang") {

return "Selamat $waktu, $nama!";

}

Penjelasan

function : Deklarasi Function

salam : Nama Function

$nama = "Admin" : Parameter 1 dengan nilai default "Admin"

$waktu = "Datang" : Parameter 2 dengan nilai default "Datang"

return : Nilai yang dikembalikan;


Cara pemanggilannya:

salam()
