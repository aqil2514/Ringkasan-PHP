1.1 Dasar-dasar PHP (Sintaks)

Sama dengan HTML, PHP memilikis sintaks dasar untuk membuka dan menutup kode, yaitu dengan "<?php" untuk membuka dan "?>" untuk menutup, contoh:


<?php echo "ini adalah kode di dalam PHP" ?>



Code tersebut masih bisa disederhanakan menjadi berikut


<?= echo "ini adalah kode di dalam PHP" ?>


1.2 Dasar-dasar PHP (Variabel)

Untuk menambahkan suatu data dalam variabel, dapat dibuat dengan tanda dolar, contoh.

$nama = "Muhamad Aqil Maulana";

Variabel tidak boleh diawali dengan angka. Namun, boleh mengandung unsur angka.

Cara memanggilnya, cukup buka tag PHP dan langsung tulis nama variabelnya sesuai dengan yang ditentukan.

<?php echo $nama?>

1.3 Dasar-dasar PHP (Penggabungan dua string)

Penggabungan dua string dalam PHP menggunakan tanda titik. Contoh, saya memliki tiga variabel.

$nama_depan= "Muhamad";

$nama_tengah= "Aqil";

$nama_belakang= "Maulana";


Di bawah ini adalah contoh pemanggilannya.

$nama_depan= "Muhamad" . " " $nama_tengah= "Aqil" . " ". $nama_belakang= "Maulana"

1.5 Dasar-dasar PHP (Komentar)

Ada dua cara memberika komentar dalam PHP, yaitu dengan tanda kurung dua kali "//" dan tanda bintang ditambah tanda kurung "/*...*/", contoh:

2.1. Control Flow


Dalam konteks ini, tidak beda jauh antara Javascript dengan PHP. PHP memiliki 2 jenis Control Flow, yaitu pengulangan dan pengkondisian. Berikut adalah Keywordnya:

  1. for
  2. while
  3. do...while
  4. if...else

2.2. Control Flow (for)


For digunakan untuk menentukan berapa kali suatu kode itu dijalankan. Keyword ini seringkali digunakan untuk membuat kode yang berulang-ulang. format:

for ($nilai_awal; $nilai_awal </> nilai akhir; increment / decrement )


Contoh:

<?php

for ($x = 0; $x < 10; $x++ ){

echo "The number is: $x <br>";

}

?>


Penjelasan:

$x = 0; // Variabel awal adalah 0

$x < 10; // Variabel akhir adalah 10

$x++ // Increment

2.3. Control Flow (While)


While adalah sebuah jenis control flow yang akan menjalankan serangkaian kode apabila kondisi di dalam tanda kurung itu bernilai true. Format

while (kondisi)


Contoh :

while ($i < 5){

echo "Hello World <br>";

$i++ // Increment

}

2.4. Control Flow (Do...While)

Do...While adalah salah satu control flow PHP yang berfungsi untuk melakukan looping juga. Bedanya dengan While, Do While akan melakukan minimal satu kali pengerjaan sedangkan while tidak.

Sintaks:

Do{

Aksi

} while(kondisi)


Contoh:

$i = 0;

do{

echo "Hello World <br>";

$i++

} while($i < 5 )

2.5. Control Flow (If...Else)

Sama seperti yang lainnya, If...Else adalah looping yang ada pada PHP. Tidak hanya If...Else, PHP juga mengenal istilah elseif. Penulisannya bisa disambung (elseif) dan bisa dipisah (else if).

Sintaks:

if(Kondisi) {

Aksi

} elseif(Kondisi) { // Juga bisa ditulis else if

Aksi

} else{Aksi}

3.1. Function

Sama seperti Javascript, PHP juga mengenal function. Dalam PHP, ada dua jenis Function, yaitu:

  1. Built In Function : Function yang telah tersedia oleh PHP secara default dapat digunakan secara bebas.
  2. User Defined Function : Function yang dibuat sendiri.

Di bawah ini adalah beberapa Built In Function:

  • String

    1. .strlen() = Menghitung panjang string
    2. .strcmp() = Menggabungkan dua buah string
    3. .explode() = Menggabungkan kedua string menjadi array
    4. .htmlspecialchars() = Memberikan keamanan pada website

    Utility

    1. .isset() = Apakah variabel telah dibuat? (Hasil boolean)
    2. .empty() = Variabel apa yang kosong?
    3. .die() = Memberhentikan program
    4. .sleep() = Memberhentikan sementara program

    Date / Tiime

    1. .time()
    2. .date()
    3. .mktime()
    4. .strtotime()
    5. Lain-lain

3.2. User Defined Function

Dalam PHP, programmer bisa membuat functionnya tersendiri. Programmer bisa menggunakan format seperti di bawah ini.

function salam($nama = "Admin", $waktu = "Datang") {

return "Selamat $waktu, $nama!";

}

Penjelasan

function : Deklarasi Function

salam : Nama Function

$nama = "Admin" : Parameter 1 dengan nilai default "Admin"

$waktu = "Datang" : Parameter 2 dengan nilai default "Datang"

return : Nilai yang dikembalikan;


Cara pemanggilannya:

salam()

4.1. Array

Array dalam PHP adalah variabel yang dapat memiliki banyak nilai dan nilainya itu boleh berbeda-beda tipe.

Ada dua unsur utama dalam array, yaitu key dan value. Key adalah kunci utama dari sebuah array. Sedangkan value adalah nilai dari keynya.

Key itu terbagi menjadi dua, yaitu index key dan associative key. Secara default, key dalam array adalah index dan dimulai dari 0.


Perhatikan contoh di bawah ini:

Array dengan cara lama:

$hari = array("Senin", "Selasa", "Rabu");

Array dengan cara baru:

$bulan = ["Januari", "Februari", "Maret"]

Array dengan tipe data yang berbeda:

$arr1 = [123, "tulisan", false]


Keterangan:

  • $hari, $bulan, dan $array merupakan sebuah variabel.

  • Dalam cara lama, programmer harus menuliskan keterangan array dan dikurung dengan tanda kurung biasa "(...)"

  • Dalam cara baru, programmer sudah tidak harus menuliskan keterangan. Cukup menandakan array dengan tanda kurung siku sebagaimana variabel $bulan dan $arr1

  • Pada variabel $hari, array dengan index 0 memiliki key senin. Begitupun juga berlaku pada variabel $bulan dan $arr1

4.2. Menampilkan Array

Ada beberapa cara menampilkan Array, yaitu dengan cara menggunakan keyword var_dump dan print_r. Echo tidak dapat menampilan array.


Contoh :

$hari = array("Senin", "Selasa", "Rabu");

$bulan = ["Januari", "Februari", "Maret"]

$arr1 = [123, "tulisan", false]

Memanggil Array

var_dump($hari);

print_r($bulan);


Seperti yang tertulis sebelumnya, echo memang tidak dapat memanggil Array. Namun, jika hanya ingin menampilkan suatu nilai dari array index tertentu, echo dapat digunakan. Contoh:

echo $arr1[1];

echo $bulan[1];

4.3. Menambahkan value baru pada array

PHP memperbolehkan para programmernya untuk menambahkan nilai atau value baru pada suatu array. Berikut adalah contohnya.

$hari = array("Senin", "Selasa", "Rabu"); // Array awal

$hari[] = "Kamis"; // Menambahkan satu value array baru

4.4. Pengulangan pada Array

Pada 2.2, kita telah mempelajari salah satu control flow for yang berfungsi untuk melakukan looping atau pengulangan.

Jika ingin melakukan pengulangan pada array, for ini tidak dapat kita gunakan.

Pengulangan pada dapat menggunakan foreach dengan sintaks dasar sebagai berikut:

foreach( $variable_array as $new_variable )


Contoh:

$hari = array("Senin", "Selasa", "Rabu"); // Array awal


foreach( $days as $day );

5.1. Array Associative

Pada 4.1, telah disinggung bahwasanya dalam PHP, ada array associative. Array Associative memungkinkan programmer untuk memberi nama sendiri key yang berada di dalam suatu array.

Sintaks pembuatannya:

$namaArray = [ "namaKey" => "valueKey" ]


echo $namaArray["namaKey"] // Cara memanggil Array Associative

6.1. Tips I (Gaya Template)

PHP itu dapat disatukan dalam HTML. Oleh karena itu, penulisan sintaks dengan gaya template ini dapat membantu.

Penulisan gaya template ini berlaku pada control flow yang telah kita pelajari seperti for, if, foreach, dsb.

Penulisannya pun sangat mudah. Ubah tanda kurung kurawal buka "{" menjadi tanda titik dua ":" dan ubah kurung kurawal tutupnya "}" menjadi endnamacontrolflow, contoh:


Menggunakan gaya biasa:

for($x = 0; $x < $y.length; $x++){

Kode HTML

}


Menggunakan gaya template

<?php for($x = 0; $x < $y.length; $x++): ?>//Berubah menjadi tanda titik dua ":"

Kode HTML

<?php endfor ?> // Berubah menjadi endfor

7.1. Macam-macam variabel

Ada beberapa jenis variabel dalam PHP, yaitu variabel lokal, variabel global, dan variabel superglobal.

Variabel lokal adalah variabel yang berada di dalam tanda kurung kurawal, biasanya sering digunakan dalam pembuatan function.

Variabel global adalah variabel yang berada di dalam teks php dan dapat diakses dari dalam dengan menggunakan keyword global.

Variabel superglobal adalah yang sudah tersedia secara superglobal dan tidak perlu menggunakan keyword tertentu dalam pemanggilannya.


$x = 10; //Variabel global

function tampilX (){

$y = 9; // Variabel lokal

global $x;; //Perlu keyyword global jika ingin mengakses variabel $x

}

7.2. Variabel Superglobal

Variabel superglobal adalah variabel yang sudah tersedia di dalam PHP dan sudah tidak perlu lagi didefinisikan.


Perhatikan tabel di bawah ini:

Nama Variabel Fungsi
$_GET Menangkap data yang ada di dalam URL. Biasa digunakan dalam pembuatan form.
$_POST Menangkap data yang dikirimkan oleh form dengan method post.
$_REQUEST Menyimpan gabungan nilai dari variabel $_GET, $_POST, dan $_COOKIE yang saling berhubungan dengan data yang dikirim bersamaan dengan request user
$_SESSION Menyimpan data user yang login dalam kurun waktu tertentu. Secara default, waktunya adalah 1440 detik atau 24ment.
$_COOKIE Menyimpan data user yang pernah login dalam device user itu sendiri. Biasa digunakan saat ingin menerapkan auto login.
$_SERVER Menyimpan segala informasi tentang request yang ditangkap oleh server.
$_ENV Berisi data tentang environment yang skrip PHP berjalan di atasnya
$_FILES Menangkap file yang digunakan dalam elemen input tipe file pada HTML
$GLOBALS Menyimpan semua variabel global yang dijalankan saat program diluncurkan.
